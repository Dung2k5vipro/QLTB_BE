# SKILL: validate

Muc tieu: Validate dau vao ngay tai tang request.

Quy tac:

- Validate tach rieng cho `body`, `query`, `params`.
- Validation file chi dinh nghia rule, khong chua business logic.
- Middleware `validate` nhan schema va tra loi 400 khi sai dinh dang.
- Thong bao loi ngan gon, de frontend xu ly.
- Dat ten schema theo hanh dong: `createX`, `updateX`, `listX`.
